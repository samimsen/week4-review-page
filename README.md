# TailwindCSS ile Landing Page - [Canlı site için tıklayın](https://samimsen.github.io/week4-review-page/)

Bu proje, **TailwindCSS** ve **Bootstrap Icons** kullanılarak oluşturulmuş **responsive bir landing page** projesidir. Sayfa; navigasyon barı, hero bölümü, müşteri yorum kartları, iletişim formu ve sosyal medya linkleri içeren bir footer barındırır.

## Kullanılan Teknolojiler

- HTML5
- **TailwindCSS** (CDN üzerinden)  
- **Bootstrap Icons** (CDN üzerinden)
